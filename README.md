# QA Portfolio

## 概要

QA学習用に作成した、PythonとSeleniumによるログイン画面のテスト自動化サンプルです。

## 使用技術

* Python
* Selenium
* HTML

## テスト内容

* 正常ログインテスト
* パスワード誤入力テスト
* 空欄ログインテスト

## 実行方法

1. Pythonをインストール
2. Seleniumをインストール
3. main.py を実行

## 学んだこと

* Seleniumによるブラウザ自動操作
* 要素取得と入力処理
* 成功/失敗判定の自動確認

## 補足

* 実行結果を確認しやすいよう、各テストケース間に短い待機時間を設定しています。
* 実行終了時は Enter キー入力でブラウザを閉じる仕様です。
